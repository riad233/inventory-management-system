- `.sidebar-toggle-btn` - Toggle button
- `.nav-item` - Navigation items
- `.nav-item.active` - Current page

### JavaScript
- `initializeSidebarState()` - Load on page load
- `toggleSidebar()` - Handle toggle click
- `closeSidebarMobile()` - Close on mobile
- `handleWindowResize()` - Responsive handling

### CSS Properties
- `position: fixed` - Always visible
- `transform: translateX()` - Smooth animation
- `transition: 0.35s cubic-bezier(...)` - Easing
